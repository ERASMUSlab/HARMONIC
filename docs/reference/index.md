# Package index

## All functions

- [`HARMONIC_ANALYSIS()`](https://erasmuslab.github.io/HARMONIC/reference/HARMONIC_ANALYSIS.md)
  : Run the HARMONIC analysis pipeline
- [`HARMONIC_COMPARE()`](https://erasmuslab.github.io/HARMONIC/reference/HARMONIC_COMPARE.md)
  : Compute HARMONIC tree scores across multiple cases
- [`HARMONIC_COMPARE_SINGLE()`](https://erasmuslab.github.io/HARMONIC/reference/HARMONIC_COMPARE_SINGLE.md)
  : Compute HARMONIC tree scores for a single case
- [`HARMONIC_HeatMap()`](https://erasmuslab.github.io/HARMONIC/reference/HARMONIC_HeatMap.md)
  : Draw a heatmap of HARMONIC results
- [`HARMONIC_PREPROCESSING()`](https://erasmuslab.github.io/HARMONIC/reference/HARMONIC_PREPROCESSING.md)
  : Run GO enrichment preprocessing for HARMONIC
- [`HARMONIC_SEPERATE()`](https://erasmuslab.github.io/HARMONIC/reference/HARMONIC_SEPERATE.md)
  : Separate HARMONIC terms by condition specificity (with PASS/FAIL
  filtering)
- [`HARMONIC_SEPERATE_forDA()`](https://erasmuslab.github.io/HARMONIC/reference/HARMONIC_SEPERATE_forDA.md)
  : Differential-activity (DA) tree filtering for selected conditions
- [`HARMONIC_SEPERATE_forMULTI_range()`](https://erasmuslab.github.io/HARMONIC/reference/HARMONIC_SEPERATE_forMULTI_range.md)
  : Select significant HARMONIC terms within a condition range
- [`HARMONIC_TERMLISTING()`](https://erasmuslab.github.io/HARMONIC/reference/HARMONIC_TERMLISTING.md)
  : Build the HARMONIC term tree list across multiple cases
- [`HARMONIC_TERMLISTING_single()`](https://erasmuslab.github.io/HARMONIC/reference/HARMONIC_TERMLISTING_single.md)
  : Build the HARMONIC term tree list for a single case
- [`HARMONIC_TERM_PLOT_forSINGLE()`](https://erasmuslab.github.io/HARMONIC/reference/HARMONIC_TERM_PLOT_forSINGLE.md)
  : Plot HARMONIC terms for a single tree (bar or chord)
- [`HARMONIC_TERM_barPLOT_forMULTI()`](https://erasmuslab.github.io/HARMONIC/reference/HARMONIC_TERM_barPLOT_forMULTI.md)
  : HARMONIC multi-case term bar plot (term-level)
- [`HARMONIC_TREE_PLOT_forSINGLE()`](https://erasmuslab.github.io/HARMONIC/reference/HARMONIC_TREE_PLOT_forSINGLE.md)
  : Plot HARMONIC trees for a single condition
- [`HARMONIC_TREE_cirPLOT_forMULTI()`](https://erasmuslab.github.io/HARMONIC/reference/HARMONIC_TREE_cirPLOT_forMULTI.md)
  : HARMONIC multi-case circular plot (tree-level)
- [`theme_dose()`](https://erasmuslab.github.io/HARMONIC/reference/theme_dose.md)
  : HARMONIC default plotting theme
